# Lift Control App
## Prerequisites

- [.NET SDK 6.0+] (or higher)
- A C# compatible editor (Visual Studio, Visual Studio Code etc.)

## Clone the repository
- git clone https://github.com/vemulurin/LiftController.git
- cd LiftController
## Build the project
dotnet build .\LiftController.sln
## Run the application
dotnet run
